# decisions 模板(会话级)

> 用法:记录会话内用户的拍板,作为实施依据与收尾报告附件。**长期决策不以本模板实例为权威**——必须由维护者转入 [`AGENTS.md §19.7 / §21`](../../../AGENTS.md) D-lock,或沉淀为批次评审稿(`docs/archive/batches/` 范式,冻结不回改)。
> 区分:**拍板**(用户明确说"按方案 A 做")≠ **默许**(用户没反对)。只有前者可登记。

```markdown
## Decisions — <会话/任务标题>(<日期>)

### D-<n>:<决策一句话>
- **背景**:<为什么需要拍板;证据(路径:行号)>
- **选项**:方案 A <…> / 方案 B <…>
- **拍板结果**:<方案 X;用户原话或可追溯位置(会话时间点/PR 评论链接)>
- **影响面**:<模块 / 测试 / 文档>
- **回退条件**:<什么情况下此决策需要重开>
- **是否需要转长期 D-lock**:✅(建议进 AGENTS §21 追加)/ ❌(仅本任务有效)
```
