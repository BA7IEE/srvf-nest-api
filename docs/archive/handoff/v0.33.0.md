## 1. TL;DR

- **v0.33.0 = 招新实名 OCR 鉴伪版充分利用(1 feat)+ 2 修复** — 把腾讯云身份证鉴伪版 `RecognizeValidIDCardOCR` 从「只榨一点点」做到**充分利用**,并修复同链路两处 bug。**仅大陆身份证路径增强**(护照/回乡证维持现状);**0 端点 / 0 RBAC 码 / 0 BizCode 变更**,唯一 schema 触点 = `recruitment_applications` +6 列 additive nullable(第 31 migration)。
- **3 笔(#458 / #460 / #461)**:
  - **招新实名 OCR 充分利用鉴伪版**(#461,**Added**,档 D):请求体显式带 **7 `Enable*` 开关**(仅 mainland)取回扩展字段 + 字段级反光/不完整标志 + 顶层裁剪图 base64;`mapResponse` 扩展映射嵌套 `IDCardInfo.{Sex,Nation,Birth,Address,Authority,ValidDate}.Content`。**recognize 端顾问式扩展回显**(`+ocrDetail`:字段级性别/民族/出生/住址/签发机关/有效期 + `documentType` + 卡片级 `cardWarnings`;**纯回显不进任何判定**——`recognized` 清晰度 / S4b 六分流 / `riskLevel` / `antiForgeryWarnings` 行为锁一律不破)。**`recruitment_applications` +6 列 additive nullable 无 enum**(`ocrAddress`/`ocrNation`/`ocrAuthority`/`ocrValidDate` 顾问式存档 + `idCardCropImageKey`/`idCardPortraitImageKey`);submit 时主体/头像裁剪图解码为两独立 blob 入库,缺省 → 列 null 不阻断。**admin 取证件照扩三图签名 URL**(原图 + 主体裁剪 + 头像裁剪;**不新增端点**;敏感新列纳入既有 `read.sensitive` 分级)。**`birthDate`/`genderCode` 推导零改动**(仍由身份证号推导权威,**OCR 的 Sex/Birth 仅回显、不持久、不覆盖**)。明文 PII 永不入日志(L3)。
  - **大陆身份证 OCR 恒返「证件照不清晰」修复**(#460,**Fixed**,档 C):鉴伪版响应把姓名/证件号**嵌套**在 `IDCardInfo.{Name,IdNum}.Content`,而 `mapResponse` 旧按标准 `IDCardOCR` 顶层字符串读 → 恒 `undefined` → `recognized:false` → 整条大陆身份证自动核验死(恒 `ocr_unclear` 重拍)。修法 = **①** 映射改读嵌套 Content + `WarnInfos` 收窄(仅复印/翻拍/PS 计防伪);**②去混淆** —— `IDCardInfo` 容器整块缺失 = 契约/系统错 → 抛 `RealnameApiError`(识别端 `27031` / 提交端 `ocr_error`),不再静默降级;**③** dev 安全调试日志(`logger.debug`,生产默认不输出,**全程无 PII**)。**0 schema / 0 migration / 0 RBAC / 0 BizCode / 0 新端点 / 契约 snapshot 零变**(`27031` 本就在声明错误集内)。
  - **招新报名 admin 列表过滤参 400 修复**(#458,**Fixed**,档 C):`GET admin/v1/recruitment/applications` 的 `cycleId`/`statusCode`/`riskLevel` 经 loose `@Query('x')` 绕开 DTO → 全局 `forbidNonWhitelisted` 误拒 **400**。修法 = 新建 `RecruitmentApplicationListQueryDto extends PaginationQueryDto`(三参 `@IsOptional` + `riskLevel @IsIn` 复用既有常量),controller 改 `@Query() dto`、去 loose 旁路(**不动全局安全设置**)。**0 schema / 0 migration / 0 RBAC / 0 BizCode / 0 新端点 / `EXPECTED_ROUTES` 零变**;contract snapshot 受控微调(三参 `required:true→false` 矫正 + `riskLevel` 增 enum + 补 description)。
