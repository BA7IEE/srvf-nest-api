- **运维侧**:三条运维通道仍休眠/按需配置(实名 OCR / 微信 / SMS);**微信订阅上线须**配 `wechat_settings` + 经 admin 端点填各类型真实 `templateId` + 按真实模板字段名核对内置字段映射;**短信兜底上线须**配 `sms_settings`(providerType=TENCENT_SMS + 凭证)+ 填真实 `templateIdNotification`(零变量「请打开 App 查看」模板须先过审)。SMS 数据 retention 仍走手动 SQL SOP(**不解锁 cron**)。
- 本版**不**解锁新 cron、**不**做报名前 openid 非会员推送、**不**自建全员短信异步批处理基建。
